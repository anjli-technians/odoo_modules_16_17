Import and Manage Bank Statements

This app makes the importation process of bank statements very easy through a few clicks. Whether it is multiple sheet file formats or different types of statements, the tool allows for efficient management and importation of them. There is no need to hassle around with manual entry of data, as the application streamlines the workflow to work with all sorts of formats smoothly. We also offer sample sheets to help you refer to them easily. With this app, it has never been easier and faster to manage your bank statements.

This app lets you import a Bank Statement in a few simple clicks

Create Statement with Create Statement button

Wizard opens for entering the Statement Details

Reference is a Mandatory field to create Bank Statement

Fill the necessary details and click on Save button

Note that the list appears empty as we have not imported the bank statement file yet.

Click on IMPORT STATEMENTS to do so

You can download samples of Bank Statements

or you can select a .csv or .xlsx file and import straightaway

If any other file format is selected an error message appears

Select any supported file format and click on IMPORT

Your Bank Statement appears in the list

The Total Amount and End Balance will always remain same

You can select SAVE AND IMPORT

The Import file wizard appears
